QuantCoderFS

Modular AI workflows for alpha research in finance. Local-first. No login. No cloud.

Chat with Fundamentals

Transforms financial queries into structured research notes using EODHD data and LLMs. Includes financial ratios analysis, OHLCV history and news filtering.

Search Articles

Uses LLM-enhanced agents to expand user queries and search arXiv and CrossRef. Returns matched papers with title, abstract, and source metadata.

Summary Generation

Extracts and restructures academic papers into concise, sectioned Markdown summaries. Suitable for research documentation and export to Notion.

Code Generation

Converts structured research summaries into QuantConnect-ready Python strategies. Includes syntax validation, logic modeling, and agent-based code refinement.
